du

```shell
du = Disk Usage: 用于查看文件或目录的大小。
df用于查看磁盘或分区使用情况的命令
du用于按照指定容量单位来查看文件或目录在磁盘中的占用情况

# 以易读的容量格式显示指定目录内各个文件的大小信息
du -h /etc

# 以易读的容量格式显示指定目录内总文件的大小信息
du -sh /Dir

# 显示指定文件的大小信息（默认单位为K）
du File.cfg
```



df

```shell
df命令来自英文词组“report file system disk space usage”的缩写，其功能是用于显示系统上磁盘空间的使用量情况。df命令显示的磁盘使用量情况含可用、已有及使用率等信息，默认单位为Kb

# 带容量单位的显示系统全部磁盘使用量情况：
df -h 

# 带容量单位的显示指定磁盘分区使用量情况： 
df -h /boot 

# 显示系统中所有文件系统格式为XFS的磁盘分区使用量情况：
df -t xfs 
```
